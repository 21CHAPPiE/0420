# 滩坑水库调度规则摘要

1. 台汛期 156.5m < Z <= 161.5m 时，按补偿凑泄方式调度。
2. 梅汛期和枯水期 160.0m < Z <= 161.5m 时，按补偿凑泄方式调度。
3. 161.5m < Z <= 161.7m 时，按 6000m3/s 控制流量下泄。
4. Z > 161.7m 时，溢洪道闸门全部打开，机组参与泄洪。
5. Z > 165.27m 时，开启泄洪洞参与泄洪，发电暂停。

## 闸门顺序
- 开启顺序: #1 -> #6 -> #3 -> #4 -> #2 -> #5
- 关闭顺序: #5 -> #2 -> #4 -> #3 -> #6 -> #1
